# Day-09: Simple Calculator Project
## Project Objective 
The objective of this project is to build a simple command-line calculator using Python that can perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The calculator is designed to allow continuous calculations using previous results, improving user interaction and making the program more dynamic.

## What i learnt
1. **Functions with Return Values:** Learned how functions process inputs and return results that can be stored and reused.
2. **Multiple Return Values**: Learned how a function can return more than one value for further use.
3. **Docstrings:** Learned how to document functions using triple quotes to improve code readability.
4. **Recursion:** Learned how a function can call itself to repeat a process when needed.
5. **Function Mapping:** Learned how to use dictionaries to connect operators with their corresponding functions.
5. **Loops and Input Validation:** Learned how to control program flow with loops and validate user input to create an interactive calculator.
## How It Works
1. The program starts by displaying a welcome message.
2. The user is prompted to enter the first number.
3. A list of available mathematical operations (+, -, *, /) is displayed.
4. The user selects an operation and enters the next number.
5. The program performs the calculation using a dictionary that maps symbols to functions.
6. The result is displayed in a readable format.
7. The user is asked if they want to continue:
    - If yes, the result becomes the new starting number and calculation continues.
    - If no, the program stops and displays the final result.
