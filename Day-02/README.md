# Day-02:BILL TIP CALCULATOR
## Project Objective
The objective of this project is to develop a Bill Split Calculator that determines how much each person should pay by dividing the total bill among a group, while also including an optional tip percentage in the calculation.
## What I Learned
1. Data Type and Conversion:I learnt about Strings,Integers,Floats,Boolean data types and how to convert them
2. Mathematical Operation:It helps  to perform math using simple symbols called arithmetic operators.
3. Formatted string:which means injecting variables or expressions into a piece using f-string

## How Bill Tip Calculator Works
* Enter bill amount, tip percentage, and number of people.
* Calculate the tip and add it to the bill.
* Divide the total by the number of people.
* Display each person’s share.
