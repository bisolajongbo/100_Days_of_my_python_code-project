# Day-01:Band Name Generator
## Project Objective
TThe objective of this project is to create a simple Python program that generates a fun and unique band name by combining the user’s city name and pet name. The project also helps beginners practice Python concepts such as user input, variables, and output formatting.
## What I Learned
*  How to collect user input in Python using input()
* How to store data in variables
* How to combine strings to create a result
* How to display output using the print() function
* Basic understanding of Python syntax and program flow
* How to build a simple beginner-friendly Python project
* Understanding that yellow highlights in code editors usually indicate warnings, suggestions, or syntax issues that may need attention but do not always stop the code from running.


## How Band Name Generator Works
1. The program asks the user to enter the name of the city they grew up in.
2. It then asks the user to enter the name of their pet.
3. The program combines both inputs together.
4. Finally, it displays a fun and unique band name suggestion to the user.
